# Humor Mechanism Notes

A Mechanism-Chain Approach for Chinese Verbal Humor（适用于LLM的中文幽默机制链）

一个关于中文冷笑话结构的机制化分析笔记

副标题：尝试用“机制链”而不是传统术语描述语言幽默

---

## 项目简介

作者：**Chun Wen** · [ORCID: 0009-0002-6376-8984](https://orcid.org/0009-0002-6376-8984)

这个仓库不是完整的幽默理论，也不是正式学术论文。它更像是一组“机制笔记”。

本项目尝试：

- 将语言幽默拆解为可观察、可复用的结构链条
- 使用偏“过程/机制”的描述方式，而非仅做传统人文学术术语归类
- 探索这种表示法是否可能作为一种分析工具，用于LLM时代的人机协同语言分析

核心贡献：将一个中文冷笑话拆解为可复用的六步机制链，并提供LLM可读的JSON映射。

目前的核心案例是：

> “穿拖鞋更舒服告诉我们一个道理：决定鞋好不好穿的关键是鞋好不好脱。”

# 核心案例拆解

## 原句

> 穿拖鞋更舒服告诉我们一个道理：决定鞋好不好穿的关键是鞋好不好脱。

---

# 第一层：语言形式上的“伪对称”

句子利用了中文中的形式对称：“好穿”与“好脱”。

由于“穿/脱”在字面上形成反义关系，句子会自然产生一种错觉：它们似乎处在同一个评价维度中构成对立。

但在实际语义中：

- “好穿”更接近：穿着体验是否舒适
- “好脱”更接近：动作是否方便、阻力是否低

两者并不构成严格逻辑反义。

因此这一层的作用是：

> 用语言形式制造“对称性假设”，但该假设在语义层面并不成立。

---

# 第二层：现实层面的“模糊中介”

这里的关键变量是**拖鞋**。

拖鞋具有一种边界性质：

- 它属于“鞋”
- 但在体验上接近“接近光脚”的状态

因此它压缩了“穿”和“脱”之间的操作差异：

- 穿 ≈ 半脱
- 穿鞋 ≈ 接近无鞋状态

这一中介变量使得“脱”的特征开始部分参与“穿”的评价解释，从而让前一层的语言错觉在经验层面获得部分支撑。

---

# 第三层：隐藏的经验前提

该结构背后隐含一个常见经验判断：

> 在多数情境下，光脚状态比穿鞋更舒适。

由此形成一个隐含的评价参考轴：

```text
光脚（参考极值状态）
↑
越容易脱的鞋
↑
越接近该状态的鞋
```

在这个结构中，“好脱”在某种程度上成为“接近舒适状态”的代理指标。

---

# 三层结构如何共同作用

整体效果可以理解为：

```text
语言形式对称（制造假设）
→ 现实中介压缩边界（拖鞋）
→ 经验评价轴提供收敛方向
```

最终形成一种结构效果：

> 表面上像悖论，但在经验层面是局部成立的，因此产生幽默感。

---

# 一个有意思的现象

如果将所有隐含条件完全展开（例如明确“光脚更舒服”这一前提），那么原本的张力会明显减弱。

这说明该类幽默依赖于：

> 维度未完全显式化 + 局部信息压缩

---

# 核心机制链（Mechanism Chain）

可以将该过程压缩为如下结构：

```text
语言形式对称
→ 诱导对立假设
→ 引入模糊中介变量
→ 评价关注点发生偏移
→ 经验背景完成局部收敛
```

---

# 面向LLM的结构化映射（Logic Mapping）

为便于机器或提示词使用，可将其映射为结构化状态流：

```json
{
  "S0 (初始态)": "建立语言形式对称（A vs B）",
  "V1 (中介项)": "引入边界模糊的中介变量（拖鞋）",
  "Action (作用)": "压缩A/B之间的操作与体验边界",
  "Shift (变化)": "评价重点从A转移到B的代理属性",
  "S1 (收敛态)": "引入经验背景完成局部自洽",
  "Output (结果)": "表面悖论 + 局部成立 = 幽默效果"
}
```

# 当前观察补遗（LLM行为观察的经验记录）

在使用该机制链作为结构化提示（prompt schema）测试不同大语言模型时，可以观察到一些稳定的行为模式。这些现象目前仍属于经验性观察，而非严格结论。

## 1. 向上逃逸：宏大解释倾向

部分模型在面对局部机制断点时，可能倾向于引入更高层级的抽象理论进行补全，例如社会学或认知科学层面的解释。这种行为可以理解为：

> 在缺乏局部机制约束时，系统倾向于用更高抽象层级维持解释连续性。

## 2. 向内收缩：局部机械化解释倾向

另一种常见现象是模型被限制在局部变量内部时，会过度细化因果链条，形成高度机械化但经验感较弱的解释路径。

> 在缺乏外部评价锚点时，解释可能过度局部化。

## 3. 一个可能的解释方向（非结论性）

上述两种现象可能与以下因素相关：

- 对具身经验锚点的依赖程度不同
- 对隐含评价基准（implicit baseline）的定位方式不同

这些仍然是开放性问题，尚未形成严格理论。

# 适用范围与边界

本笔记目前主要适用于以下语境：

- 中文语言环境（依赖词语形式对称与语义省略）
- 具身经验参与的日常类幽默文本

对于其他语言体系或高度抽象符号系统，其直接迁移效果尚未验证。
